## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/igson660/DesafioInCicle-Inc.-.git

# Access
$ cd DesafioInCicle-Inc.

# Install dependencies
$ yarn / npm install

# Run the project
$ yarn start / npm start

# The server will initialize in the <http://localhost:3000>
```
